## 📝 开发指南

### 环境准备
1. 安装HBuilderX或VS Code + uni-app插件
2. 配置微信开发者工具（小程序开发）
3. 配置Android Studio（Android开发）
4. 安装Node.js和npm

### 开发命令
```bash
# 安装依赖
npm install

# 开发模式 - 微信小程序
npm run dev:mp-weixin

# 开发模式 - Android应用
npm run dev:app-plus

# 构建发布
npm run build:mp-weixin  # 构建微信小程序
npm run build:app-plus   # 构建Android应用
```

### 开发规范
- 使用ESLint进行代码规范检查
- 遵循Vue.js官方编码规范
- 使用语义化的Git提交信息
- 编写必要的注释和文档
